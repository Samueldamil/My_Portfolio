Weather App

This project involves building a full-stack Weather App using HTML, CSS, JavaScript, Node.js, and Express.js.
The frontend provides a responsive and user-friendly interface that allows users to search for weather information by city. The backend, built with Node.js and Express.js, acts as a secure proxy to the OpenWeatherMap API, protecting the API key and handling data requests.
Together, the app fetches and displays real-time weather details such as temperature, condition, and location in a clean and interactive layout.
