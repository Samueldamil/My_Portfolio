PostBook

This project involves building a mini social media platform called PostBook, where users can create, view, update, and delete posts containing text and images.
The backend is powered by Node.js, Express, and MongoDB, with Cloudinary integrated for secure image storage.
The frontend is developed using HTML, CSS, and JavaScript, offering a clean and responsive user interface.
The project demonstrates full-stack development, RESTful API integration, file upload handling, and deployment on Render.

To-Do List App

This project involves building a responsive To-Do List App using React, HTML, and CSS.
It allows users to add, delete, and mark tasks as completed.
The app also uses localStorage to ensure tasks are saved even after refreshing or closing the browser.

Weather App

This project involves building a full-stack Weather App using HTML, CSS, JavaScript, Node.js, and Express.js.
The frontend provides a responsive and user-friendly interface that allows users to search for weather information by city. The backend, built with Node.js and Express.js, acts as a secure proxy to the OpenWeatherMap API, protecting the API key and handling data requests.
Together, the app fetches and displays real-time weather details such as temperature, condition, and location in a clean and interactive layout.
